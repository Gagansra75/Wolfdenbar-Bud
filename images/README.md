## How to Add Your Images:

1. Download all the images from your live website at wolfdenbar.com
2. Place them in this `/images/` folder
3. Make sure the filenames match exactly as listed above
4. The website will automatically load them

## Downloading Images from Your Live Site:

You can download images from wolfdenbar.com by:
1. Right-clicking on each image
2. Select "Save Image As..."
3. Save with the exact filename as shown above

Or use browser developer tools:
1. Open wolfdenbar.com
2. Press F12 or right-click and select "Inspect"
3. Go to the Network tab
4. Filter by "Img"
5. Refresh the page
6. Right-click on each image and select "Open in new tab"
7. Save each image

## Image Specifications:

- **Gallery Images**: Recommended size 800x600px or higher
- **Slideshow Images**: Recommended size 1200x800px or higher
- **Hero Video**: MP4 format, recommended 1920x1080px or higher
